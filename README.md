* Task#2 -  For Twins: 2. Math operations 8 KUY
```javascript
function iceBrickVolume(radius, bottleLength, rimLength) {
  const rimWidth = bottleLength - rimLength;
  return (rimWidth * (radius * 2)) * radius;
}
```
